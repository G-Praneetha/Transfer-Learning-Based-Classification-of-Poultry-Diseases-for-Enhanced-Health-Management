Introduction

This website provides access to a smart, AI-powered diagnostic system for classifying and managing common poultry diseases like Salmonella, Newcastle Disease, and Coccidiosis, helping improve health outcomes and productivity.

Project Objectives

  • Provide fast, accurate poultry disease diagnosis using AI
  • Enable farmers to manage poultry health independently
  • Deliver real-time treatment and prevention guidance
  • Educate veterinary students with hands-on case studies

Technology Stack

  • Machine Learning: TensorFlow / PyTorch
  • Transfer Learning: ResNet, MobileNet
  • Mobile Application: Flutter / React Native
  • Backend Services: Firebase / AWS

Use Case Scenarios

Rural Community Outbreak

Farmers use the mobile app to input symptoms and receive a diagnosis of Coccidiosis, enabling rapid treatment and containment.

Commercial Farm Monitoring

A large poultry farm uses the system for daily health checks. The system flags potential Newcastle Disease early, allowing swift action.

Veterinary Student Training

Students use real-life datasets via the app to diagnose and learn treatment protocols for poultry diseases.

Future Scope

  • Include additional poultry diseases
  • Support real-time environmental sensor data
  • Expand to multilingual interface for global use
